# 危险化学品经营单位主要负责人刷题网站 - 执行计划

## 项目概述
基于上传的题库文件（849道题：482道单选 + 367道判断），构建一个刷题网站。
**核心需求**：字大、鼠标点点就能用、适合刷题练习。

## 技术栈
- React + TypeScript + Vite + Tailwind CSS（使用 vibecoding-webapp-swarm 技能）
- shadcn/ui 组件库
- 纯前端静态网站，localStorage 存储进度

## 功能规划
1. **首页/模式选择**：大按钮选择刷题模式（顺序练习、随机练习、模拟考试）
2. **顺序练习**：按题型分类，逐题作答，即时反馈
3. **随机练习**：打乱题目顺序，随机出题
4. **模拟考试**：随机抽100题，限时作答，交卷评分
5. **答题界面**：大字显示题目和选项，点击选择/判断，显示正误
6. **进度统计**：记录正确率、已做题数、收藏错题
7. **错题本**：自动收录错题，可反复练习

## 执行阶段

### Stage 1 — 题库数据解析
- 将原始 .doc 文件内容解析为结构化 JSON 数据
- 单选题：提取题干、选项A/B/C/D、正确答案
- 判断题：提取题干、正确答案（正确/错误）
- 输出：questions.json

### Stage 2 — 网站构建（vibecoding-webapp-swarm）
- 读取技能文件，按照技能规范构建
- 构建刷题应用的所有组件和页面
- 集成题库数据
- 输出：完整可运行的 webapp

### Stage 3 — 部署
- 部署为静态网站
